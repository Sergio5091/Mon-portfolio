# Portfolio Frontend

## Overview

React frontend application for a personal portfolio website using modern web technologies.

## Stack

- **Framework**: React 18 + Vite
- **Styling**: Tailwind CSS + Radix UI components
- **Routing**: Wouter
- **State Management**: React Query for data fetching
- **Animation**: Framer Motion
- **TypeScript version**: 5.9
- **Package manager**: pnpm

## Key Commands

- `pnpm --filter @workspace/portfolio run dev` — start development server
- `pnpm --filter @workspace/portfolio run build` — build for production
- `pnpm --filter @workspace/portfolio run serve` — preview production build

See the portfolio package for component structure and development details.
